# Dialog-if Brew

## How do I install these formulae?

`brew install dialog-if/brew/<formula>`

Or `brew tap dialog-if/brew` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dialog-if/brew"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
